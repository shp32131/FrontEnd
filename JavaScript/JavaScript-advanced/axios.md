# axios

## 什么是axios?   
> axios是一个基于Promise的HTTP库，是用Promise对ajax的封装,可以用于浏览器和Node.js中   

**特性**
- 从浏览器中创建XMLHttpRequests  
- 从Node.js创建http请求  
- 支持Promise API  
- 拦截请求和响应  
- 转换请求数据和响应数据  
- 取消请求
- 自动转换JSON数据
- 客户端支持防御XSRF
